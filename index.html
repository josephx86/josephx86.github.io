Registration

Going back about a week in time, I registered for the program on Oct 7, 2015. So I guess that was my 'first day of school'. I immediately started the Edx Hardvard CS50 course.

HarvardX: CS50x3 Introduction to Computer Science

Oct 14, 2015

You can check more details about CS50 here. So far the course seems to be an introduction to programming?? At least that is how I see it. Week 0 introduced me to Scratch, which is a really cool and nifty programming environment. Coming from languages like C#, C, PHP, Java, and so on, and then using Scratch, I have to say that I now appreciate the power and capabilities of those languages. While Scratch is easy to learn and understand (it literally takes a few minutes to get the hang of it), it is also somewhat crippled when you are coming from programming tools that are not as restrictive. In any case, Scratch makes programming fun and enjoyable. I am just not sure how I can use Scratch in a real-world scenario, but I'm sure with time I will figure it out... I mean, it has only been a week since I knew the language/tool even existed.

I enjoyed Week 1 more than 0 because the tutorials focused on good old C language. Yes, it was just the basic hello-world bits but all the same, it felt good to be using familiar tools. So far I like the course and I will see how things shape up as time goes.
